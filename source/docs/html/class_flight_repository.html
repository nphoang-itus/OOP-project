Detailed Description

Lớp repository để quản lý các thao tác cơ sở dữ liệu cho thực thể Flight.

Lớp này cung cấp các phương thức để thực hiện các thao tác CRUD (Create, Read, Update, Delete) trên bảng flights trong cơ sở dữ liệu MySQL. Ngoài ra còn hỗ trợ quản lý ghế ngồi và các truy vấn đặc biệt cho chuyến bay.
