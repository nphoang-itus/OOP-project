Detailed Description

Helper struct cung cấp thông tin chi tiết về các lỗi ContactInfo.

Struct này chuyển đổi enum error thành chuỗi mã lỗi và thông báo có thể đọc được, đồng thời cung cấp phương thức tiện lợi để thêm lỗi.
