Detailed Description

Value object đại diện cho thông tin liên lạc đầy đủ của hành khách.

ContactInfo đóng gói email, số điện thoại và địa chỉ của hành khách. Value object này đảm bảo tính hợp lệ của từng thành phần theo các chuẩn quốc tế:

  • Email theo RFC 5321 (tối đa 254 ký tự)
  • Số điện thoại theo E.164 (tối đa 15 chữ số)
  • Địa chỉ tùy chọn (tối đa 100 ký tự)
