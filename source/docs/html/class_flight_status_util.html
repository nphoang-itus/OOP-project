Detailed Description

Lớp utility cung cấp các phương thức chuyển đổi cho FlightStatus.

FlightStatusUtil cung cấp các phương thức để chuyển đổi giữa enum FlightStatus và các biểu diễn chuỗi khác nhau (tiếng Anh và tiếng Việt) để hiển thị và lưu trữ dữ liệu.
