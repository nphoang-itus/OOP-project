Detailed Description

Đại diện cho số hộ chiếu cùng với quốc gia cấp phát.

Lớp này là một value object đóng gói số hộ chiếu cùng với mã quốc gia cấp phát. Nó cung cấp các khả năng xác thực, phân tích và định dạng cho số hộ chiếu.
