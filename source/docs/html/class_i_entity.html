Detailed Description

Lớp cơ sở trừu tượng cho tất cả các thực thể domain.

Interface này định nghĩa hợp đồng chung mà tất cả các thực thể domain phải thực hiện. Nó cung cấp cả định danh cấp độ cơ sở dữ liệu (qua ID số nguyên) và định danh cấp độ nghiệp vụ (qua ID chuỗi) để hỗ trợ cả lưu trữ và logic domain.

Tất cả các thực thể được mong đợi là các value object bất biến với phương thức factory để tạo và hỗ trợ nhân bản để sao chép an toàn.
