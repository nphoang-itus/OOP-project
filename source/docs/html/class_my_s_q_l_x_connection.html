Detailed Description

Hiện thực lớp IDatabaseConnection bằng MySQL X DevAPI.

Sử dụng giao thức X Protocol để kết nối và thao tác với MySQL Server. Cung cấp đầy đủ chức năng truy vấn, statement, transaction và kiểm soát lỗi.

Lớp này implements Singleton pattern để đảm bảo chỉ có một connection active trong toàn bộ application. Điều này giúp:

  • Tránh connection leak
  • Quản lý resource hiệu quả
  • Đảm bảo transaction consistency

Đặc điểm kỹ thuật:

  • Thread-safe với std::mutex
  • Connection pooling thông qua singleton
  • Automatic resource management với RAII
  • Comprehensive error handling với Result<T>
  • Support cho prepared statements với parameter binding
  • Full transaction control (ACID properties)
