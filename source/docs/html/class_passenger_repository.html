Detailed Description

Lớp repository để quản lý các thao tác cơ sở dữ liệu cho thực thể Passenger.

Lớp này cung cấp các phương thức để thực hiện các thao tác CRUD trên bảng passengers trong cơ sở dữ liệu MySQL. Bao gồm cả các truy vấn đặc biệt để tìm kiếm hành khách theo số hộ chiếu.
