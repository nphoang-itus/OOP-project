Detailed Description

Value object đại diện cho số serial duy nhất của máy bay.

AircraftSerial đóng gói số serial máy bay và đảm bảo tính hợp lệ của nó. Số serial phải tuân theo định dạng cụ thể: 2-3 chữ cái theo sau bởi 1-7 chữ số. Value object này là bất biến và cung cấp các phương thức factory để tạo instance an toàn.
