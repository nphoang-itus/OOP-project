Detailed Description

Lớp dịch vụ chính chịu trách nhiệm quản lý các thao tác liên quan đến vé máy bay.

Lớp này áp dụng các nguyên tắc SOLID và các pattern thiết kế để cung cấp một interface đầy đủ cho việc quản lý vé, đặt chỗ, tìm kiếm và xác thực nghiệp vụ.
