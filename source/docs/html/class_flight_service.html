Detailed Description

Lớp dịch vụ chịu trách nhiệm quản lý các thao tác liên quan đến chuyến bay.

Lớp này cung cấp các chức năng CRUD cơ bản và các thao tác nghiệp vụ phức tạp cho việc quản lý chuyến bay, bao gồm quản lý ghế ngồi, lịch trình, trạng thái bay.
