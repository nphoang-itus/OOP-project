Typedef Documentation

◆ Result

template<class T>
using Result = std::expected<T, CoreError>

Kiểu Result có thể chứa giá trị kiểu T hoặc CoreError.

Template alias này cung cấp cách tiếp cận hàm số để xử lý lỗi sử dụng std::expected. Các thao tác trả về Result<T> thay vì ném exception, cho phép xử lý lỗi rõ ràng và kiểm soát luồng tốt hơn.

◆ VoidResult

using VoidResult = std::expected<void, CoreError>

Kiểu Result cho các thao tác không trả về giá trị

VoidResult đại diện cho kết quả của các thao tác có thể thành công (không có giá trị trả về) hoặc thất bại với CoreError.
