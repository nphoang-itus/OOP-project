Detailed Description

Lớp utility cung cấp các phương thức chuyển đổi cho TicketStatus.

TicketStatusUtil cung cấp các phương thức để chuyển đổi giữa enum TicketStatus và các biểu diễn chuỗi khác nhau (tiếng Anh và tiếng Việt) để hiển thị và lưu trữ dữ liệu, cũng như parsing từ database.
