Detailed Description

Thu thập và quản lý nhiều lỗi validation.

ValidationResult cho phép tích lũy nhiều lỗi validation trong quá trình validation, cung cấp cái nhìn tổng thể về tất cả các lỗi validation thay vì dừng lại ở lỗi đầu tiên. Điều này đặc biệt hữu ích cho validation form và các tình huống validation đối tượng phức tạp.
